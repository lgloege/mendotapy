Mendotapy Documentation
======================================================

Mendotapy is a Python 3.8+ package that provides easy acces to analysis-ready ice phenology data from Lake Mendota.

The goal of this package is to make it easier to analyze this awesome data.


Installation
------------

.. code-block:: bash

   # for latest release
   pip install mendotapy

   # for bleeding-edge up-to-date commit
   pip install -e git+https://github.com/lgloege/mendotapy.git
